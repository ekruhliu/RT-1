# RT
OpenCL ray tracing with refraction and reflection support(no recursion).
<br />
## My features:
 * refraction / reflection
 * fresnel effect
 * figures
 * lights: ambient, point, directional, spot
 * color light
 * phong effect and specular
 * light absorbing
 * shadow
 * color shadow on transparent object
 * anti aliasing
